## 2024-05-24 - Semantic Form Accessibility
**Learning:** For radio button groups, using `<div>` and `<label>` to group and describe the set of options is insufficient for screen readers. Using `<fieldset>` and `<legend>` provides the correct semantic grouping, allowing assistive technologies to announce the question when the user focuses on any of the radio options.
**Action:** Always prefer `<fieldset>` and `<legend>` for grouping related form controls like radio buttons or checkboxes to ensure optimal accessibility.
